# 🚀 Инструкция по запуску Python API для новичков

Эта инструкция поможет вам запустить серверную часть (API) вашего дипломного проекта, даже если вы никогда раньше не работали с Python.

---

## 🛠 Шаг 1: Подготовка (что нужно установить)

Прежде всего, убедитесь, что на вашем компьютере установлены:
1.  **Python 3.10 или новее**: [Скачать с официального сайта](https://www.python.org/downloads/). 
    *   *Важно:* При установке обязательно поставьте галочку **"Add Python to PATH"**.
2.  **Microsoft ODBC Driver for SQL Server**: Он нужен для связи Python с вашей базой данных. [Скачать здесь](https://learn.microsoft.com/ru-ru/sql/connect/odbc/download-odbc-driver-for-sql-server). Выбирайте версию x64.

---

## 📂 Шаг 2: Открытие проекта

1. Скачайте файлы из ветки `feature/python-api`.
2. Откройте папку `SchoolScheduleAPI` в вашем любимом редакторе кода (например, **VS Code**) или просто через проводник.

---

## ⚙️ Шаг 3: Установка зависимостей (библиотек)

Откройте терминал (командную строку) в папке `SchoolScheduleAPI` и выполните команду:

```bash
pip install -r requirements.txt
```

Эта команда автоматически скачает и установит все необходимые "двигатели" для нашего API (FastAPI, SQLAlchemy и др.).

---

## 🗄 Шаг 4: Настройка базы данных

По умолчанию API настроено на подключение к локальной базе данных SQL Server, которую создавал ваш C# проект. 

Если ваша база данных называется иначе или находится на другом сервере, откройте файл `app/core/database.py` и измените строку:

```python
SQLALCHEMY_DATABASE_URL = "mssql+pyodbc://(localdb)\\MSSQLLocalDB/School11_Schedule_DB?driver=ODBC+Driver+17+for+SQL+Server&trusted_connection=yes"
```

---

## 🚀 Шаг 5: Запуск!

В терминале, находясь в папке `SchoolScheduleAPI`, введите:

```bash
python main.py
```

Если всё сделано правильно, вы увидите сообщение:
`INFO: Uvicorn running on http://0.0.0.0:8000`

---

## 🔗 Как проверить работу?

1.  **Главная страница**: Откройте в браузере [http://localhost:8000](http://localhost:8000). Вы должны увидеть приветственное сообщение.
2.  **Интерактивная документация (Swagger)**: Перейдите по адресу [http://localhost:8000/docs](http://localhost:8000/docs). 
    *   Здесь вы увидите список всех доступных команд (эндпоинтов).
    *   Вы можете нажать кнопку **"Try it out"**, а затем **"Execute"**, чтобы получить данные прямо из вашей базы данных.

---

## ❓ Что делать, если не работает?

*   **Ошибка "pip не является внутренней или внешней командой"**: Вы забыли поставить галочку "Add Python to PATH" при установке Python. Переустановите его.
*   **Ошибка подключения к БД**: Убедитесь, что ваш SQL Server запущен и база данных `School11_Schedule_DB` существует.
*   **Порт 8000 занят**: Если на этом порту уже что-то работает, вы можете изменить порт в самом конце файла `main.py`.
