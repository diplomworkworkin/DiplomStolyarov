# SchoolScheduleDiplom

Дипломный проект: модуль формирования и автоматизации расписания для МБОУ СШ №11 г. Волгодонска.

## Быстрый запуск
1. Откройте `SchoolSchedule.sln` в Visual Studio 2022.
2. Убедитесь, что стартовый проект — **SchoolScheduleApp**.
3. Проверьте строку подключения в `SchoolSchedule/Context/SchoolDbContext.cs` (используется LocalDB).
4. Примените миграции (если база пуста):
   ```
   dotnet ef database update --project SchoolSchedule --startup-project SchoolScheduleApp
   ```
5. Запустите приложение (F5).

## Тестовые учетные записи
- **Админ**: `admin / admin`
- **Учитель**: `teacher1 / teacher1`

## Запуск тестов
```
dotnet test
```

## Ручной чек-лист (10–25 пунктов)
1. Запуск приложения без ошибок (окно входа открывается).
2. Некорректный логин/пароль показывает предупреждение.
3. Вход под админом открывает панель администратора.
4. Вход под учителем открывает кабинет учителя.
5. Вход под учеником открывает кабинет ученика.
6. Кнопка «Выйти» возвращает на окно авторизации.
7. Авто-составление расписания отрабатывает без ошибок.
8. При отсутствии нагрузки выводится понятное сообщение.
9. В расписании администратора корректно отображаются уроки класса.
10. В кабинете учителя отображается личное расписание.
11. Фильтр «День недели» ограничивает список уроков учителя.
12. Фильтр «Класс» в расписании учителя работает корректно.
13. Фильтр «Предмет» в расписании учителя работает корректно.
14. Очистка фильтров возвращает список ко всем урокам.
15. Для ученика расписание отображается для своего класса.
16. При отсутствии расписания показан пустой список без ошибок.
17. Смена темы (если есть) не ломает читаемость текста.
18. Данные в таблицах читаемы в темной теме.
19. Кнопки подсвечиваются при наведении.
20. Переходы между страницами плавные (анимации).
21. При закрытии окна приложение завершается корректно.
